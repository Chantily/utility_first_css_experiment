#IDK B2B frontend


### 1. install gulp

`npm install gulp -g`

### 2. Install dependency

`npm install`

### 3. Run it

`gulp default`


### Additional: Rebuild Tailwind CSS

`npm run refresh`

### Additional: Rebuild Tailwind CSS 

`npm run build`. 

This will create `style.min.js` and now you can refer from your HTML files instead of `style.css`
